Features
Dashboard Overview: Get a quick snapshot of all projects with statistics and recent activity

Project Management: Create, edit, and delete projects with detailed information

Task Tracking: Add, update, and complete tasks within projects

Task Form: User-friendly form for creating and editing tasks

Project Details: Comprehensive view of individual projects with progress tracking

Settings Panel: Customize application preferences including notifications, appearance, language, privacy, and accessibility

Responsive Sidebar: Collapsible navigation with mobile support and dark mode toggle

Prerequisites
Node.js (v14 or higher)

npm or yarn package manager

Clone the repository:

git clone [<your-repository-url>](https://github.com/FarzamN/ReactForAppicoder.git)
cd ReactForAppicoder

Install dependencies:
npm install

Start the development server:
npm run dev
